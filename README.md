# Base Ten Blocks — интерактивная доска

Визуальный инструмент для учителей: блоки разрядов (единицы, десятки, сотни, тысячи), drag-and-drop, таблица разрядов, перегруппировка, режимы десятичных и сравнения, аннотации, **глобус и карта**, экспорт PNG. **Всё работает в браузере** — без базы данных, Redis и внешних map API.

## О проекте

Сайт предназначен для демонстрации на интерактивной доске или проекторе. Учитель перетаскивает блоки, показывает разряды, перегруппировку и сравнение чисел. Приложение — статический Next.js-сайт, который можно бесплатно разместить на Vercel.

## Требования

- **Node.js** 18 или новее
- **npm** 9+ (или yarn / pnpm)
- Современный браузер: Chrome, Edge, Firefox, Safari
- Для интерактивной доски: поддержка touch (планшет, SMART Board и т.п.)

Проверка версии:

```bash
node -v
npm -v
```

## Локальный запуск

### 1. Клонирование

```bash
git clone <url-репозитория>
cd "Tools in visual teaching"
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Режим разработки

```bash
npm run dev
```

Откройте в браузере: **http://localhost:3000**

На экране вы увидите:
- боковую панель приложений: **Блоки** и **Карта** (глобус);
- переключатель режимов (Целые числа / Десятичные / Сравнение);
- палитру блоков в toolbar;
- цветную таблицу разрядов (слева) и рабочую область (справа);
- внизу — итоговое число и разбивку по разрядам.

Горячая перезагрузка включена: изменения в коде сразу видны в браузере.

### 4. Продакшен-сборка локально

```bash
npm run build
```

Результат — статические файлы в папке `out/`.

Проверка собранного сайта без Vercel:

```bash
npx serve out
```

Откройте указанный адрес (обычно http://localhost:3000).

### 5. Линтинг

```bash
npm run lint
```

## Деплой на Vercel

Переменные окружения **не требуются**. Серверная часть не используется.

### Шаг 1 — Репозиторий

Загрузите проект в GitHub, GitLab или Bitbucket:

```bash
git add .
git commit -m "Base Ten Blocks interactive board"
git push origin main
```

### Шаг 2 — Импорт в Vercel

1. Войдите на [vercel.com](https://vercel.com).
2. Нажмите **Add New… → Project**.
3. Выберите репозиторий с этим проектом.
4. Vercel определит **Framework Preset: Next.js** автоматически.

### Шаг 3 — Настройки сборки

| Параметр | Значение |
|----------|----------|
| Build Command | `npm run build` |
| Output Directory | `out` (определяется Next.js static export) |
| Install Command | `npm install` |
| Environment Variables | не нужны |

### Шаг 4 — Deploy

Нажмите **Deploy**. Через 1–2 минуты сайт будет доступен по адресу вида `https://ваш-проект.vercel.app`.

### Шаг 5 — Обновления

Каждый push в ветку `main` (если включён auto-deploy) автоматически пересобирает сайт.

### Опционально: свой домен

В Vercel: **Project → Settings → Domains** — добавьте домен и следуйте инструкциям по DNS.

## Использование доски

| Действие | Как |
|----------|-----|
| Добавить блок | Перетащите с палитры на доску или в таблицу разрядов |
| Переместить | Режим «Выбор» → перетащите блок |
| Выбрать один блок | Один клик по блоку |
| Мультивыбор | Двойной клик добавляет блок к выделению |
| Удалить | Выберите блок(и) → «Удалить» или клавиша Delete |
| Собрать | Выделите 2+ одинаковых блока → частичный блок (3 из 10) или полная перегруппировка при 10+ |
| Собрать (колонка) | Без выделения: 10 мелких в колонке → 1 крупный |
| Разъединить | Выберите блок (в т.ч. частичный) → «Разъединить» |
| Цвет ручки/маркера | При активной ручке или маркере — палитра цветов в toolbar |
| Рисование | Ручка / Маркер / Текст (кнопка OK) / Ластик-кисть (стирает только пройденную часть линии) |
| Полный экран | Кнопка «Экран» (для проектора) |
| Скачать снимок | Кнопка «PNG» |
| Десятичные | Режим «Десятичные» + сетка 10×10 (клик по клеткам) |
| Сравнение | Режим «Сравнение» — сверху только счётчики и миниатюры (при count > 0), без иконок при нуле; блоки в нижнем workspace A/B; внизу знак >, < или = |

## Глобус и карта

Переключитесь на **Карта** в боковой панели слева.

| Функция | Как |
|---------|-----|
| Плоская / 3D | Кнопки «Плоская» и «Глобус» в toolbar |
| Физическая / геополитическая | Переключатель стиля карты |
| Рисование | Те же инструменты и цвета, что на доске блоков |
| Перетаскивание | Физическая: континенты с реальными контурами; геополитическая: отдельные страны; движение с учётом zoom карты |
| Эффект пазла | На месте убранной части остаётся пунктирный контур; отпускание рядом с контуром возвращает кусок на место (snap) |
| Подписи | Континенты (RU); в геополитическом режиме — адаптивно: крупные страны при отдалении, все при сильном приближении; размер в экранных пикселях (~9–18px) растёт с zoom |
| Масштаб (плоская) | Кнопки +/− и «Сброс» в toolbar; колесо мыши при инструменте «Выбор» |
| Глобус (геополит.) | Цветные страны, контурные границы, крупные светлые подписи; детальная текстура и геометрия 50m |
| Сброс | «Сбросить карту» / «Очистить рисунки» / сброс масштаба |

Данные карты — статические файлы в `public/geo/` (`world-countries-50m.json`, `continents-110m.json`, текстура Земли `earth-topography.jpg`). При сборке `npm run build` копируются страны 50m и при необходимости загружается высокодетальная текстура глобуса. Континенты генерируются скриптом `npm run build:geo`. Внешние tile-серверы и map API **не используются**.

## Возможности

- Блоки: единица, десяток и сотня — плоские 2D-квадраты; тысяча — стопка из 10 изометрических 3D-пластин (пропорции 1 : 10 : 100 : 1000)
- Частичное объединение: 3 единицы → «десяток» с 3 заполненными и 7 призрачными ячейками
- Мультивыбор блоков (клик / двойной клик)
- Drag-and-drop с поддержкой touch
- Цветная таблица разрядов: счётчик, иконка типа, подсветка готовности к сборке
- Разбивка числа (например: `2Т 3С 4Д 5Е`)
- Режим десятичных с сеткой
- Режим сравнения: счёт по workspace, миниатюры вверху, drop только в нижний сектор
- Аннотации: ручка, маркер с выбором цвета, текст, сегментный ластик-кисть
- Карта-пазл: континенты и страны с призрачными контурами; drag с учётом zoom и snap при возврате; zoom на плоской карте; геополитический глобус
- Тёплый UI (шрифт Nunito, кремовый фон)
- Экспорт доски в PNG (только в браузере)
- Полноэкранный режим
