# Base Ten Blocks — интерактивная доска

Визуальный инструмент для учителей: блоки разрядов (единицы, десятки, сотни, тысячи), drag-and-drop, таблица разрядов, перегруппировка, ручка, маркер и текст. Работает полностью в браузере — без базы данных и внешних сервисов.

## Локальный запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Сборка

```bash
npm run build
```

Статические файлы появятся в папке `out/`.

## Деплой на Vercel

1. Импортируйте репозиторий на [vercel.com](https://vercel.com).
2. Framework Preset: **Next.js** (определяется автоматически).
3. Переменные окружения не требуются.
4. Deploy.

## Возможности

- Перетаскивание блоков с палитры на доску и в таблицу разрядов
- Автоматический подсчёт числа
- **Собрать** — 10 мелких блоков → 1 крупный (с переносом в следующий разряд)
- **Разобрать** — выбранный блок → 10 мелких
- Инструменты: выбор, ручка, маркер, текст
- Полноэкранный режим для интерактивной доски

## Стек
